Este código HTML crea una página básica que muestra una imagen llamada "foto.jpg".

Explicación del código: : Es la etiqueta raíz que contiene todo el contenido de la página web. : La sección donde usualmente se incluyen metadatos, enlaces a estilos, scripts, y otros elementos que no son visibles directamente en la página. En este caso, está vacío. : Contiene el contenido visible de la página web. : Muestra una imagen en la página web. src="foto.jpg": Especifica el archivo de la imagen que se mostrará. Debe estar en el mismo directorio que el archivo HTML. alt="pintura geometrica": Define un texto alternativo que se muestra si la imagen no se carga.
